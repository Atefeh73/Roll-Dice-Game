# Roll-Dice-Game with Python:

A simple dice rolling game built with **Python**. The user chooses **how many times to roll the dice**, and the program **generates random results**.

What I learned: 
  Using **while** and **for** loops
  Handling user input
  Using the random module
  Basic error handling with **try/except**
